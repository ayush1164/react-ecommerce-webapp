# 🛍️ Mini E-Commerce Web App

A responsive and modern mini e-commerce web application built using React.js. The application allows users to browse products, search, filter by category, sort by price, add items to the cart, and complete a frontend checkout flow.

---

## 🚀 Features

- 🛒 Add products to cart
- ➕ Increase product quantity automatically
- 🔍 Search products by name
- 🏷️ Filter products by category
- ↕️ Sort products by price (Low → High & High → Low)
- 💳 Frontend checkout page
- ✅ Order success page with tracking ID
- 📱 Responsive product grid
- 🎨 Modern UI with hover effects
- ⚡ Fast image loading using optimized local assets
- 🦶 Footer with company links and social media icons

## ⚙️ Installation

Clone the repository

git clone https://github.com/ayush1164/react-ecommerce-webapp.git
Move into the project directory

cd mini-ecommerce-web-app
Install dependencies

npm install
Run the development server

npm run dev
Open

http://localhost:5173
